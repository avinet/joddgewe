JoddGewe
=========
Et program for å konvertere og vise styringsfiler til ortofoto. 
Programmet støtter (foreløpig) konvertering til og fra JGW/TFW, MMM og SOSI. Programmet er fritt tilgjengelig for alle som måtte ha bruk for det
